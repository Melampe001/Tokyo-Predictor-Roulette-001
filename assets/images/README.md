# Assets Directory

This directory contains app assets for Tokyo Roulette.

## Required Assets

### App Icon
- `icon.png` - 1024x1024 PNG for iOS and store listing
- Generated automatically for Android via adaptive icons

### Splash/Branding
- `logo.png` - TokyoApps® logo
- `splash_background.png` - Splash screen background (optional)

## Directory Structure

```
assets/
├── images/
│   ├── icon.png          # App icon
│   ├── logo.png          # TokyoApps® logo
│   └── splash_bg.png     # Splash background
├── fonts/                # Custom fonts (optional)
└── animations/           # Lottie/Rive animations (optional)
```

## Asset Guidelines

### Icon
- Size: 1024x1024 pixels
- Format: PNG with transparency
- Content: Casino/roulette wheel motif
- Brand: Include TokyoApps® styling

### Screenshots (for Play Store)
Required resolutions:
- Phone: 1080x1920 (portrait) or 1920x1080 (landscape)
- 7" Tablet: 1200x1920
- 10" Tablet: 1600x2560
